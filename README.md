# Truco Paulistinha – Marcador de Pontos

## Summary  
Truco Paulista é um marcador de pontos moderno feito para partidas de Truco Paulistinha. A aplicação permite registrar pontuações, acompanhar o histórico das mãos e determinar automaticamente o vencedor da partida, oferecendo uma experiência rápida, intuitiva e responsiva.

---

## Features
- Marcação de pontos para Truco Paulistinha
- Dois times: Time A vs Time B
- Valores de mão: 2, 4, 6, 9 e 12 pontos
- Registro automático do histórico de todas as mãos
- Finalização automática da partida ao atingir 12 pontos
- Interface leve, moderna e responsiva
- Navegação fluida entre telas (Home → Partida)
